# TES3MP Admin GUI

## Description
This program was made to be able to modify player data for TES3MP while that player is offline.

## Features
- Modify, add, or delete player data , i.e.
  - Faction Expulsions
  - Server Staff Rank
  - All Player Skills and Attributes

## Built-with
### Backend Server
- NodeJS
- ExpressJS

### Frontend Application
- React
- Tailwind CSS
- Javascript
